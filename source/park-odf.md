## 课堂学习内容

| 内容            | 语言                                                    |
| ------------- | ----------------------------------------------------- |
| 打招呼的方式        | *Hello!*                                              |
| 问候方式          | - *How are you?*<br>- *I’m good / great / wonderful.* |
| 询问姓名和回答       | - *What’s your name?*<br>- *I’m …*                    |
| 感知数字 1–4      | one, two, three, four                                 |
| 自然拼读字母 Aa, Bb | - 字母的读音<br>- 字母在单词中的发音<br>- 字母的书写步骤<br>- 首字母的单词例子     |
